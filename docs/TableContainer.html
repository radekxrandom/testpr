onClickSortByField

Sort entries by a given field when it is clicked. On second click reverse the order. Arrow variable is passed as a prop to TableComponent and specifies field in which arrow is displayed and it's orientation. Pointing downwards if field is sorted ascending, and upwards otherwise. After sorting it calls the method to paginate newly sorted data.
